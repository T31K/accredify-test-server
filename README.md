#### Server
I'm  using Ruby on Rails for quick development and easy API access as I'm only using 2 endpoints.

##### Prerequisites 
- Ruby version: `2.6.3`
- Postgesql is installed
- Server is running on `http://localhost:3000/`
```
$ cd server
$ bundle install
$ rails s
```
